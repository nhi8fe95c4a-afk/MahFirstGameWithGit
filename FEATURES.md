# Особенности проекта MahFirstGameWithGit

## Обзор
Это 2D-игра на Unity (версия 6000.0.40f1) с использованием Universal Render Pipeline (URP).

## Реализованные функции

### 1. Управление персонажем (PlayerController2D)
Основной скрипт управления игроком включает следующие функции:

#### Движение
- **Плавное горизонтальное движение**: Персонаж перемещается влево/вправо с настраиваемой скоростью
- **Параметры движения**:
  - `moveSpeed`: Скорость движения (по умолчанию 8 единиц)
  - `acceleration`: Ускорение при начале движения (60 единиц)
  - `deceleration`: Замедление при остановке (70 единиц)

#### Прыжки
Реализована продвинутая система прыжков с несколькими улучшениями:
- **Обычный прыжок**: Нажатие пробела или кнопки Jump
- **Двойной прыжок**: Возможность прыгнуть повторно в воздухе (настраиваемое количество через `maxJumps`)
- **Coyote Time**: Возможность прыгнуть после схода с платформы (0.1 сек)
- **Jump Buffer**: Запоминание нажатия прыжка перед приземлением (0.1 сек)
- **Переменная высота прыжка**: 
  - Короткое нажатие = низкий прыжок (с множителем 2x)
  - Долгое удержание = высокий прыжок
- **Улучшенная физика падения**: Быстрее падение с множителем 2.5x
- **Воздушные прыжки слабее**: Прыжки в воздухе имеют множитель силы 0.9x от обычного прыжка

#### Проверка земли
- **Автоматическое создание GroundCheck**: Система автоматически создает точку проверки касания земли
- **Визуальная отладка**: Гизмо показывает состояние (зеленый = на земле, красный = в воздухе)
- **Настраиваемые параметры**:
  - Радиус проверки (0.15 единиц)
  - Маска слоев для определения земли

### 2. Система ввода (Input System)
Проект использует новую Unity Input System с предустановленными действиями:

#### Действия игрока
- **Move**: Движение (Vector2) - для WASD/стрелок/стика геймпада
- **Jump**: Прыжок (кнопка)
- **Look**: Осмотр (Vector2) - для мыши/правого стика
- **Attack**: Атака (кнопка)
- **Interact**: Взаимодействие (кнопка с удержанием)
- **Crouch**: Присесть (кнопка)
- **Sprint**: Бег (кнопка)
- **Previous/Next**: Переключение (кнопки)

#### Поддержка устройств
- Клавиатура и мышь
- Геймпад (консольный контроллер)
- Тачскрин (для мобильных устройств)

### 3. Графика и рендеринг
- **Universal Render Pipeline (URP)**: Современный конвейер рендеринга Unity
- **2D оптимизация**: Настройки специально для 2D игр
- **Volume Profiles**: Настройка постобработки

### 4. Архитектурные особенности

#### Автоматизация и удобство
- **Auto Ground Check**: Автоматическое создание точки проверки земли при добавлении компонента
- **Required Components**: Автоматическое добавление Rigidbody2D и Collider2D
- **Freeze Rotation**: Автоматическая блокировка вращения персонажа

#### Физика
- Использование `linearVelocity` (новый API Unity 6)
- Правильная настройка Rigidbody2D для 2D платформера
- Плавное движение через `FixedUpdate`

### 5. Организация проекта
- **Структура Assets**: 
  - Scenes (игровые сцены)
  - Settings (настройки проекта)
  - Scripts (скрипты игры)
- **Version Control**: Настроен .gitignore для Unity проектов
- **Шаблоны сцен**: URP 2D Scene Template

## Потенциал для расширения

### Готовые к реализации функции
Система Input уже включает привязки для:
- Атака (не реализовано в коде)
- Взаимодействие с объектами (не реализовано)
- Приседание (не реализовано)
- Бег/спринт (не реализовано)
- Переключение предметов/оружия (не реализовано)

### Возможные улучшения
1. **Система боя**: Использовать действие "Attack"
2. **Система инвентаря**: Использовать "Previous/Next" для переключения
3. **Взаимодействие с миром**: Использовать "Interact" для диалогов/предметов
4. **Анимация**: Добавить Animator для визуализации действий
5. **Звук**: Добавить AudioSource для эффектов прыжков и шагов
6. **UI**: Добавить интерфейс здоровья, очков, инвентаря

## Заключение
Проект представляет собой хорошо структурированную основу для 2D платформера с профессиональным подходом к управлению персонажем. Код использует современные практики Unity и готов к расширению дополнительными игровыми механиками.
