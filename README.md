# MahFirstGameWithGit

## Описание
2D платформер на Unity 6 с использованием Universal Render Pipeline (URP).

## Основные возможности

### Управление персонажем
- ✅ Плавное горизонтальное движение с ускорением/замедлением
- ✅ Продвинутая система прыжков (Coyote Time, Jump Buffer, переменная высота)
- ✅ **Двойной прыжок** (можно прыгнуть второй раз в воздухе)
- ✅ Автоматическая проверка касания земли
- ✅ Улучшенная физика падения

### Система ввода
- ✅ Поддержка клавиатуры, мыши и геймпада
- ✅ Настроенные действия: движение, прыжок, атака, взаимодействие, присесть, бег
- ✅ Использование новой Unity Input System

### Система сбора яблок
- ✅ Коллекционируемые яблоки с эффектом левитации
- ✅ Счётчик очков в правом верхнем углу экрана
- ✅ Увеличение счёта при касании яблок игроком
- ✅ Плавная анимация парения яблок вверх-вниз

### Система диалогов (отключена, но доступна)
- ✅ Исправлена возможность закрытия диалога клавишей E
- ✅ Система остаётся в проекте, но неактивна

### Технологии
- Unity 6000.0.40f1
- Universal Render Pipeline (URP)
- Unity Input System
- Rigidbody2D физика

## Как играть
- **WASD / Стрелки**: Движение
- **Пробел**: Прыжок
- **E**: Взаимодействие с объектами / Закрыть диалог
- Поддержка геймпада

## Разработка
Откройте проект в Unity 6 или новее для редактирования.

### Настройка системы сбора яблок
После открытия проекта в Unity Editor:
1. **Автоматическая настройка**: Меню → Tools → Setup Apple Collection System
2. **Ручная настройка**: Следуйте инструкциям в [APPLE_COLLECTION_SETUP.md](APPLE_COLLECTION_SETUP.md)
